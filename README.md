# Pynome

A Metronome created by `Parambir` with Python3.

Works on -
1. Linux (All Ver)
2. Windows (All Ver)
3. Mac (All Ver)

Required Dependencies:

1. `Pygame`
2. `tkinter`

Main File: `run`

Setup File: `setup`

Init: `chmod a+x *`

Website - <a href="https://sherry65-code.github.io/pynome.com">Pynome</a>

How to **run** : `./run`
